# Container Statistics

`cstat`

## Usage

```
 -c,--current              Get Current / Active Records <WIP>
 -cf,--controlfile <arg>   wip - Control File use to track run iterations
 -d,--delimiter <arg>      Record Delimiter (Cntrl-A is default).
 -e,--end <arg>            End time for retrieval in 'yyyy-MM-dd HH:mm:ss'
 -ff,--fileFormat <arg>    Output filename format.  Value must be a
                           pattern of 'SimpleDateFormat' format options.
 -h,--help                 Help
 -hdr,--header             Print Record Header
 -inc,--increment <arg>    Query Increment in minutes
 -o,--output <arg>         Output Base Directory (HDFS) (default
                           System.out) from which all other
                           sub-directories are based.
 -s,--start <arg>          Start time for retrieval in 'yyyy-MM-dd
                           HH:mm:ss'
 -sch,--schema             wip - Print Record Schema
 -ssl,--ssl                https connection
 -v,--verbose              show verbose output
```
